# 夕阳西下，余晖将老街的青砖瓦片染上一层薄薄的橘红。

你原本只是无意间路过那家旧书店，直到风吹过，门口挂着的黄铜风铃发出“丁零”一声脆响。那声音并不响亮，却像是一把生锈的钥匙，猝不及防地撞开了记忆深处那扇紧闭的门。

空气中突然弥漫起落叶与旧纸张特有的樟脑香气，那是许多年前，某个人身上常有的味道。

你的脚步不由自主地慢了下来。视线落在门廊下那把空荡荡的藤椅上，竹藤有些开裂，在晚风里微微晃动。那一瞬间，记忆里的欢声笑语、某个午后温暖的温度，连同那些早已散落人海的过往，排山倒海般涌上心头。

其实街还是那条街，风铃也只是普通的风铃，可眼前的景物一旦叠加上了旧人的影子，心底便无端生出一场海啸。原来有些人和事从未被遗忘，它们只是静静地躲在某个熟悉的角落，等待着某一次不经意的触碰，便让人红了眼眶。
